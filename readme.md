# Test Rust
Exploring Rust

## Installation
```bash
curl --proto '=https' --tlsv1.2 -sSf https://sh.rustup.rs | sh
```

## Run
```bash
cargo run
```

## Build
```bash
cargo build
```

## Test
```bash
cargo test
```

## Clean
```bash
cargo clean
```

## Check
```bash
cargo check
```

## Doc
```bash
cargo doc --open
```
